Palette data format

What color group does the 8x8 tiles uses is based on the CCC bits in YXPCCCTT:
  • CCC = 000: Palette 0
  • CCC = 001: Palette 1
  • CCC = 010: Palette 2
  • CCC = 011: Palette 3
  • CCC = 100: Palette 4
  • CCC = 101: Palette 5
  • CCC = 110: Palette 6
  • CCC = 111: Palette 7
